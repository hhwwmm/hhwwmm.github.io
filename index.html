分库分表

为什么要分库分表？

单机存储能力、连接数有限，存在性能瓶颈，数据量达到千万级以上，操作性能下降

如何分库分表？

大量数据分散到多个数据库多个表中，提升数据库性能，核心是数据切分，切分后数据快速定位与整合

垂直切分（分垂直分库和垂直分表）

  1. 垂直分库
    按照业务分类进行划分，每个业务有独立数据库
  2. 垂直分表
    基于数据表的列为依据切分，大表拆小表

Java基础

Java基础

Java 语言具有哪些特点？

  1. Java 为纯面向对象的语言。它能够直接反应现实生活中的对象。
  2. 具有平台无关性。Java 利用 Java 虚拟机运行字节码，无论是在 Windows、Linux 还是 MacOS 等其它平台对 Java 程序进行编译，编译后的程序可在其它平台运行。
  3. Java 为解释型语言，编译器把 Java 代码编译成平台无关的中间代码，然后在 JVM 上解释运行，具有很好的可移植性。
